# NASA Mission Explorer

## About the Project

This is a simple web application that shows images from space using NASA’s Astronomy Picture of the Day (APOD) API.

The idea is to let users explore space images and also check what the universe looked like on any specific date, like their birthday.

---

## Features

* View today’s space image
* Select a date to see past images
* See image title and description
* Option to view HD image

---

## API Used

NASA Open API (APOD)
https://api.nasa.gov/

---

## Tech Stack

* HTML
* CSS
* JavaScript

---

## How to Run

1. Clone the repository

```
git clone https://github.com/your-username/nasa-mission-explorer.git
```

2. Open the folder
3. Run `index.html` in your browser

---

## API Key

You can use the demo key provided by NASA, or generate your own from:
https://api.nasa.gov/
